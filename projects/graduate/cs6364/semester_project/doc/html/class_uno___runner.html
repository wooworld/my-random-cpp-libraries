Detailed Description

Runs an Uno game.

To actually play a game of Uno using the AI, create an object of this type, call setup(), then run().

Constructor & Destructor Documentation

Uno_Runner::Uno_Runner ( )

Default constructor.

Sets up m_state to have the following: UNO_NUM_PLAYERS with no cards in their hands, no names, no score, and no difficulty level. It is Player 0's turn. One unplayed deck of UNO_DECKS_PER_GAME * UNO_DECK_SIZE cards. One played deck of zero cards.

void Uno_Runner::print_state ( )

Prints out the state of the game. This includes:

  • Each player's name.
  • Each player's score.
  • Each player's hand.
  • Each player's difficulty level (if AI player).
  • The played deck of cards.
  • The unplayed deck of cards.
    Note:
    This is a very expensive operation, use sparingly!
