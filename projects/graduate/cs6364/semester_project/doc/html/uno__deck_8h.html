void create_deck ( deck &  d)

Creates and populates a deck of Uno cards.

Takes a deck and makes a new Uno deck inside of it.

Precondition:
UNO_DECKS_PER_GAME > 0
UNO_DECK_SIZE > 0
Parameters:
dA deck to store the instatiated deck in.
Note:
The previous deck's contents will be cleared.
The deck is NOT shuffled.
See also:
shuffle_deck( deck& d)

UNO_DECKS_PER_GAME directly influences the created deck. If this number is greater than 1, the created deck will have UNO_DECKS_PER_GAME regulation decks worth of cards in it. For example, if UNO_DECKS_PER_GAME = 2, then the created deck will have two standard decks worth of cards in it (for a total of 216 cards).
