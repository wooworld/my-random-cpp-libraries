Detailed Description

Holds the state of the game.

Includes the players, whose turn it is, the unplayed and played cards, and, for an AI player, the evaluation score of the state.

Member Data Documentation

unsigned int Uno_Game_State::m_eval_score

A heuristic score evaluation for this state for an AI player.

This value is rather meaningless for a real player, but for an AI player this represents how "good" a state is. A state that represents a loss for the player whose turn it is will have an extremely low value.
