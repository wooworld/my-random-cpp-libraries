Detailed Description

Contains the player, limited view of the game state.

This class contains the information for:

  • Count of cards in players' hands.
  • List of cards in this player's hand.
  • Player at play.
  • Turn count.
  • Count of cards left in unplayed deck.
  • Played deck of cards.
  • Time allowance per turn.
