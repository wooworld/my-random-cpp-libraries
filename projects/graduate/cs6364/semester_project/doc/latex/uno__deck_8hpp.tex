\subsection{\-Detailed \-Description}
\-Contains functionality for deck for the card game \-Uno. \-This file contains functions, macros, and types for creating and using a deck for a card game called \-Uno. \-The game requires matching of numeric values on cards as well as colors. \-All players draw some cards to begin and attempt to play all of their cards before other players.
