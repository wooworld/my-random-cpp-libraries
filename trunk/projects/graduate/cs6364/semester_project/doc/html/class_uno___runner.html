Detailed Description

Runs an Uno game.

To actually play a game of Uno using the AI, create an object of this type, call setup(), then run().

void Uno_Runner::print_state ( )

Prints out the state of the game. This includes:

  • Each player's name, score, level (if AI), and hand.
  • The player at play.
  • The turn count.
  • The amount of time remaining in the turn.
  • The count of and the played deck of cards.
  • The count of and the unplayed deck of cards.
    Note:
    This is an expensive operation, use sparingly!

bool Uno_Runner::remove_player ( unsigned char  i,
unsigned char  c = 0 
)

Removes player i from the game if possible.

Precondition:
0 <= i < num_players()
Parameters:
iis the player to remove from the game.
cis where to place the player's cards. Valid values are:
  • 0 = Place cards back onto bottom of the unplayed deck.
  • 1 = Place cards back onto bottom of the played deck.
  • 2 = Remove cards from the game state completely. The cards vanish into the ether.
Returns:
true If the player could be removed from the game properly.
false If the player could not be removed.

If num_players() - 1 < UNO_MIN_PLAYERS, player p will not be removed from the game and this function will return false.

The default value for c is 0.

bool Uno_Runner::setup ( )

Sets up a game to be played.

Return values:
trueIf the setup executed properly.
falseIf the setup encountered an error during execution.

Sets up the game to have:

  • UNO_NUM_PLAYERS Uno_Players. If players were added using add_player() previous to calling setup() then Uno_AI_Players are added until there are UNO_NUM_PLAYERS in the game.
  • Each Uno_Player has UNO_INIT_HAND_SIZE cards in their hands if the deck used has enough cards.
  • The DEFAULT_DECK in use for the unplayed cards unless another deck was specified using set_deck().
    Note:
    The unplayed deck is shuffled.
  • An empty played stack of cards unless another deck was specified using set_deck().
  • It is Player 0's turn.
  • It is turn 0.
