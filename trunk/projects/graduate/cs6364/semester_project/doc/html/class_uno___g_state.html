Detailed Description

Contains the full omniscient server view of the game state.

This class contains the information for:

  • List of Uno_Player (s).
  • Player at play.
  • Turn count.
  • Unplayed deck of cards.
  • Played deck of cards.
  • Time allowance per turn.
