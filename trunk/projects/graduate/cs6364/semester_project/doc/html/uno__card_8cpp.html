string card_name ( card  c)

Gets the name of an Uno card.

Parameters:
cThe card to get the name of.
Return values:
stringThe name of the card.

The card names will come back something like "R5" for a red five, or "YD2" for a yellow draw two.

Note:
A reverse card will have a type of "R". So "YR" means a yellow reverse. This is not to be confused with "R" for the color red.

unsigned char card_weight ( card  c)

Gets the weight of an Uno card.

Parameters:
cThe card to get the weight of.
Return values:
unsignedchar The weight of the card. For a regulation game, the weights are the points a player gets at the end of the game for his opponents having a given card, c, in their hand. Point values mapped according to regulation, are:
  • Number cards - Face value points (0-9 is 0-9 points, respectively)
  • Draw Two – 20 Points
  • Reverse – 20 Points
  • Skip – 20 Points
  • Wild – 50 Points
  • Wild Draw – 50 Points
