template<class T >
bool DataSet< T >::load ( const string &  path)

Loads a data set from file.

This function expects that the file located at path the format: First line of file is a comma-separated list of attribute labels. Subsequent lines of file are comma-separated lists of attribute values. These subsequent lines are the feature vectors. For example: [0] : A,B,C,D,E,Class [1] : 0,1,1,1,1,0 [2] : 1,1,0,0,0,0

Parameters:
path-- A file path to the file containing the data.
