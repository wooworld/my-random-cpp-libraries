CS347 SP2010 Puzzle Project Page - Traveling Salesman Problem (TSP)

Revision 1/12/2010. Note: it is the students' responsibility to check the CS347 website for possible new revisions!

The puzzle this semester is the classic symmetric Euclidean Travelling Salesman Problem (TSP) [see http://en.wikipedia.org/wiki/Travelling_salesman_problem], a member of the family of touring puzzles. Let n be a positive integer. Given a set of n cities and distances between each pair of cities with the distance between each pair of cities being the same in either direction (hence symmetric) and the distance between two cities corresponding to the distance being their corresponding geographic locations (hence Euclidean), find a Hamiltonian circuit with minimal total travel distance. The solution format for this puzzle is a permutation of the cities where an edge is implied between the first and last listed cities. Legal moves are those that visit a new city or, if all cities have been visited, return to the start city.

Your program should read in the file specified by the user on the command line, then generate a solution employing the algorithm specified in below phase task grid and display the resulting final solution (i.e., a solved state) following on the next line by the total travel distance. The data files that you should test on and that the grader will be using are: Problem 1, Problem 2, and Problem 3.

You should only submit a single zip file containing your source code along with any additional dependencies your program has in order to build your program, including but not limited to makefile, Microsoft Visual Studio Project file, or alternative compile script, on or before the due date specified in the below phase task grid. Your program has to be able to compile on campus standard builds; testing on these machines is therefore highly recommended! Also note that no manual configuration by the grader should be necessary to compile and execute your program; this should be handled by your makefile, MS Visual Studio Project file, or alternative compile script. Submitting is accomplished by following the instructions at http://web.mst.edu/~tauritzd/bbassignmenttool.html.
